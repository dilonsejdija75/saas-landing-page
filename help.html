Quick Start Guide

Get up and running with Boostly in under 5 minutes. This guide will walk you through the essential steps to start converting more visitors.

  1. Install the Boostly Snippet

    Contact our support team to get your custom installation code and API key.

  2. Create Your First Flow

    Start with a proven template:

    • Go to Flows in your dashboard
    • Click "Use Template" on the Welcome Flow
    • Customize the messaging for your brand
    • Set targeting rules (new users, specific pages)

  3. Test and Launch

    Before going live:

    • Use the preview feature to test your flow
    • Check on different devices and browsers
    • Activate when you're satisfied

  4. Monitor Performance

    Track your results in the Analytics dashboard:

    • Conversion rates by flow
    • User engagement metrics
    • Revenue attribution

üéâ You're all set!

Your first flow should start showing results within 24 hours. Need help? Contact our support team.
